# Профиль пользователя

Простая программа на Go, демонстрирующая работу со структурами (structs).

## Описание

Программа создает структуру пользователя с полями:
- `Name` (string) - имя пользователя
- `Age` (int) - возраст пользователя

Затем создается экземпляр структуры, заполняется данными и выводится на экран.

## Цель задачи

- Освоить работу со структурами в Go

- Научиться создавать экземпляры структур

- Практиковать вывод форматированного текста
