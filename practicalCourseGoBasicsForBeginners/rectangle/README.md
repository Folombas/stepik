# Вычисление площади фигуры

Программа на Go для вычисления площади и периметра прямоугольника.

## Описание

Программа демонстрирует:
- Работу со структурами в Go
- Вычисления с числами с плавающей точкой
- Форматированный вывод результатов

## Функциональность

- Создает структуру Rectangle с полями Width и Height
- Вычисляет площадь прямоугольника (Width * Height)
- Вычисляет периметр прямоугольника (2 * (Width + Height))
- Выводит результаты с точностью до 2 знаков после запятой

## Запуск программы

```bash
go run main.go
